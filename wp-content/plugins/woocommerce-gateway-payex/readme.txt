== DESCRIPTION ==
PayEx Gateway is a plugin that extends WooCommerce, allowing you to take payments via PayEx. This plugin utilizes the Payment Menu payment method wich means that you activate/deactivate payment methods in your PayEx account. The active payment methods will appear as available options in the PayEx payment window that the customer is redirected to when making a purchase in your store. 

== DOCUMENTATION ==
http://docs.woothemes.com/document/payex/


== IMPORTANT NOTE ==
In order to use the PayEx gateway you need a PayEx account as well as a redemption agreement with your bank. 

Note that the plugin only works if the currency is set to SEK, DKK, NOK or Euros.


== INSTALLATION	 ==
1. Download and unzip the latest release zip file.
2. If you use the WordPress plugin uploader to install this plugin skip to step 4.
3. Upload the entire plugin directory to your /wp-content/plugins/ directory.
4. Activate the plugin through the 'Plugins' menu in WordPress Administration.
5. Go WooCommerce Settings --> Payment Gateways and configure your Payson settings.
